Audio audio.getBroadcastList
Возвращает список друзей и сообществ пользователя, которые транслируют музыку в статус.
Этот метод можно вызвать с ключом доступа пользователя .
Параметры
filter определяет, какие типы объектов необходимо получить. Возможные значения :
  • friends — только друзья;
  • groups — только сообщества;
  • all — друзья и сообщества.
строка, по умолчанию all
active 1 — будут возвращены только друзья и сообщества, которые транслируют музыку в данный момент. По умолчанию возвращаются все.
флаг, может принимать значения 1 или 0
Результат
После успешного выполнения возвращает список объектов друзей и сообществ с дополнительным полем status_audio — объект аудиозаписи , установленной в статус (если аудиозапись транслируется в текущей момент).
Коды ошибок
В ходе выполнения могут произойти общие ошибки. Их описание находится на отдельной странице .
