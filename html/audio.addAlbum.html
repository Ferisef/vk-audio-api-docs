Audio audio.addAlbum
Creates an empty audio album.
This method can be called with a user token . Access rights required: audio .
Parameters
group_id Community ID (if the album will be created in a community).
positive number
title Album title.
string, required parameter
Result
Returns the ID of the created album.
Errors
302 Albums number limit is reached
Global errors can occur while running. See their descriptions on this page .
Linked methods
audio.editAlbum
Edits the title of an audio album.
audio.deleteAlbum
Deletes an audio album.
